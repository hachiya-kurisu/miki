# miki

high-stakes ride on the nightfall express

named after miki (美樹) from city hunter

## hop on

```
$ make config.h
$ vim config.h # tweak to your pleasure
$ make && make install
$ rcctl enable miki
$ rcctl start miki
```

## enjoy the view

* miki listens on ::1
* combine with relayd to share with the world
